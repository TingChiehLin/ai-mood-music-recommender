## Getting Started

### Run locally

**Backend**
```bash
cd backend
pip install -r requirements.txt
export FLASK_APP=app.py
flask run --port=5000
```

**Frontend**
```bash
cd frontend
npm install
npm run dev
```
Open [http://localhost:3006](http://localhost:3006) with your browser to see the result.
